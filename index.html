<heat-strip>

A native Web Component for 1-dimensional heatmaps

What is a Heat Strip?

A heat strip (also called a sequence map, status bar map, or timeline strip) is a 1-dimensional heatmap where each colored block represents a single unit in a series — e.g., a day, test, level, or metric sample.

The color represents the state/value: Green = OK Yellow = Warning Red = Error

Usage

<!-- Basic usage -->
<heat-strip data="ok,ok,warning,error,ok"></heat-strip>

<!-- With custom size -->
<heat-strip data="ok,ok,warning,error,ok" size="large"></heat-strip>
<heat-strip data="ok,ok,warning,error,ok" size="small"></heat-strip>

Attributes

  • data (required): Comma-separated list of states. Supported values: ok, warning, error
  • size (optional): Size of the blocks. Values: small, medium (default), large
